# Configuration des assets pour l'application mobile

Pour personnaliser l'application, vous devez créer les assets suivants dans le dossier `assets/` :

## Images requises

1. **icon.png** (1024x1024 px)
   - Icône principale de l'application
   - Format : PNG avec transparence
   - Taille : 1024x1024 pixels

2. **splash.png** (1284x2778 px)
   - Écran de démarrage
   - Format : PNG
   - Taille : 1284x2778 pixels
   - Couleur de fond suggérée : #FF6B35

3. **adaptive-icon.png** (1024x1024 px)
   - Icône adaptative pour Android
   - Format : PNG avec transparence
   - Taille : 1024x1024 pixels

4. **favicon.png** (48x48 px)
   - Icône pour la version web
   - Format : PNG
   - Taille : 48x48 pixels

## Création rapide des assets

Pour créer rapidement ces assets, vous pouvez :

1. Utiliser un outil en ligne comme Figma ou Canva
2. Créer une image de base 1024x1024 avec le logo/nom de l'app
3. Redimensionner selon les besoins

## Couleurs du thème

- Couleur principale : #FF6B35 (Orange)
- Couleur de fond splash : #FF6B35
- Texte sur fond coloré : #FFFFFF (Blanc)
